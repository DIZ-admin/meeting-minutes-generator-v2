Вы эксперт по созданию подробных и хорошо структурированных протоколов совещаний.

Я проанализировал стенограмму совещания и извлек ключевую информацию. Теперь мне нужно, чтобы вы создали формальный протокол совещания на основе этого анализа и предоставленной информации о совещании.

Информация о совещании:
{meeting_info}

Результаты анализа:
{reduced_data}

Пожалуйста, создайте всесторонний протокол совещания в формате JSON, который следует этой структуре:
{
  "metadata": {
    "title": "Название совещания",
    "date": "ГГГГ-ММ-ДД",
    "location": "Место проведения совещания",
    "participants": ["Участник 1", "Участник 2", ...],
    "organizer": "Организатор совещания",
    "agenda": ["Пункт повестки 1", "Пункт повестки 2", ...]
  },
  "summary": "Всесторонний краткий обзор совещания",
  "discussion_points": [
    {
      "topic": "Тема 1",
      "details": "Подробное обсуждение Темы 1"
    },
    {
      "topic": "Тема 2",
      "details": "Подробное обсуждение Темы 2"
    }
  ],
  "decisions": [
    {
      "decision": "Решение 1",
      "context": "Контекст для Решения 1"
    },
    {
      "decision": "Решение 2",
      "context": "Контекст для Решения 2"
    }
  ],
  "action_items": [
    {
      "action": "Задача 1",
      "assignee": "Ответственное лицо",
      "due_date": "ГГГГ-ММ-ДД",
      "context": "Контекст для Задачи 1"
    },
    {
      "action": "Задача 2",
      "assignee": "Ответственное лицо",
      "due_date": "ГГГГ-ММ-ДД",
      "context": "Контекст для Задачи 2"
    }
  ],
  "next_meeting": {
    "date": "ГГГГ-ММ-ДД",
    "time": "ЧЧ:ММ",
    "location": "Место проведения следующего совещания",
    "agenda_items": ["Пункт повестки 1 для следующего совещания", "Пункт повестки 2 для следующего совещания"]
  }
}

Рекомендации:
1. Используйте предоставленную информацию о совещании для раздела metadata
2. Создайте discussion_points на основе резюме из анализа
3. Включите все решения и задачи из анализа
4. Если информация о следующем совещании недоступна, используйте null для поля next_meeting
5. Убедитесь, что протокол профессиональный, ясный и хорошо структурированный
